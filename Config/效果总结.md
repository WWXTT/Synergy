# 效果总结（枚举类参考）

本文档用于生成卡牌效果相关的枚举类

## 1. 卡牌发动限制

| 限制类型 | 说明 | 示例 |
|----------|------|------|
| 无限制 | 无任何发动条件 | - |
| 回合限制 | 限定特定回合 | 本回合、你的回合、下个回合 |
| 时机限制 | 特定时机触发 | 当...时、在...开始时、战斗阶段 |
| 状态限制 | 特定状态条件 | 未受伤、已受伤、横置、未横置 |
| 费用限制 | 额外费用要求 | 支付X生命、消耗X法力、牺牲X |
| 目标状态 | 目标需满足条件 | 目标未受伤、目标已受伤 |

## 2. 卡牌效果

| 效果类型 | 说明 | 示例关键词 |
|----------|------|------------|
| 造成伤害 | 对目标造成伤害 | 造成X点伤害、对...造成 |
| 恢复生命 | 回复生命值 | 恢复X点生命、获得X生命 |
| 召唤/派出 | 生成单位 | 召唤、派出、置入战场 |
| 抽牌/抓牌 | 从牌库获取牌 | 抽X张牌、抓X张牌、发现 |
| 消灭/破坏 | 移除单位 | 消灭、破坏、牺牲 |
| 放逐/除外 | 完全移出游戏 | 放逐、除外、移出游戏 |
| 修改属性 | 改变数值 | +X攻击力、+X生命值、-X/-X |
| 赋予能力 | 给予特殊能力 | 获得突袭、获得嘲讽、获得飞行 |
| 返回 | 移动位置 | 返回手牌、移回战场 |
| 搜寻 | 检索牌库 | 搜寻牌库、探查 |
| 冻结 | 暂时禁用 | 冻结、横置 |
| 沉默 | 移除能力 | 沉默 |
